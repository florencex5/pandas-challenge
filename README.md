# Pandas- Challenge

There are total 15 schools selected for this dataset. Using Pandas DataFrame to analyze the school performance by creating the following summaries,
- District Summary
- School Summary
- Highest-Performing Schools by Overall Passing Percentage
- Bottom Performing Schools by Overall Passing Percentage
- Math Scores by Grade
- Reading Scores by Grade
- Scores by School Spending
- Scores by School Size
- Scores by School Type
